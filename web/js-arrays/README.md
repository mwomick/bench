# 2D vs 1D Arrays for Points

Tests for accessing 5,000,000 points in memory using a 1D array, a 2D array, and a 1D object array.

Last updated: Aug 9, 2024.

## Safari
| Test            | Time     |
|-----------------|----------|
| 1D Array        | 29.00 ms |
| 2D Array        | 57.00 ms |
| 1D Object Array | 32.00 ms |

## Chrome
| Test            | Time     |
|-----------------|----------|
| 1D Array        | 26.00 ms |
| 2D Array        | 53.10 ms |
| 1D Object Array | 46.40 ms |
